# বাংলা কোড Programming Language Documentation

## Variable Declaration and Printing

```markdown
ধরি নাম = "রিয়াসাত";
ধরি বয়স = 21;
লিখি(নাম + "এর বয়স " + বয়স);

## Conditional Statements

যদি (নাম == "রিয়াসাত") {
লিখি("হ্যালো রিয়াসাত");
} নাহলে যদি (নাম == "তারিক") {
লিখি("হ্যালো তারিক");
} নাহলে {
লিখি("তোমার নাম মিলছে না");
}

## loops

ধরি ন = 1;
অব্দি(ন < 100) {
লিখি(ন);
ন = ন + 1;
}

## Arrays and Objects

ধরি ফল = ["আম", "জাম", "কাঠাল"];
ধরি রিয়াসাত = {নাম: "রিয়াসাত", বয়স: 21};

## Functions and Classes

কাজ যোগ(অ, ব) {
ফল (অ + ব)
}

ধরি ফ = যোগ(5, 10);

ক্লাস মানুষ(নাম, বয়স, বাসস্থান) {
নাম: নাম;
বয়স: বয়স;
বাসস্থান: বাসস্থান;
}

ধরি জিশান = নতুন মানুষ("জিশান", 22, "কলকাতা");
```
